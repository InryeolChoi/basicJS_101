* 인스타그램을 클론코딩하는 것이 목표.
* 최대한 VanilaJS만을 활용해 화면 짜기.

## prac_cluster
* cluster 기능 (멀티프로세스) 연습

## prac_node
* 자바스크립트의 문법(반복문, 조건문, 자료구조 등등..)을 활용하는 것도 중요한 목표 중 하나.
* javascript 런타임인 node.js의 여러가지 기능 (특히 모듈 활용해보기)
* 가장 중요한 개념인 비동기, 콜백함수, Promises, async/await에 초점을 맞추기.
